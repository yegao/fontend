# 项目搭建

## 生成package.json

```sh
npm init
```

## 安装开发环境依赖

```sh
npm i -D webpack webpack-cli @babel/core @babel/cli @babel/preset-env @babel/preset-react
```

## 安装生产环境依赖

安装react。

```sh
npm i -S react
```

假如要使用[redux](../Redux/README.md)，安装[react-redux](../Redux/react-redux.md)。

```sh
npm i -S react-redux
```
